📄 MD → PDF API

Convert Markdown to HTML, Markdown to PDF, and HTML to PDF. One API call. Clean output.

Get Started — Free

Endpoints

POST /md-to-html

Convert Markdown to clean HTML

POST /md-to-pdf

Convert Markdown directly to a PDF file

POST /html-to-pdf

Convert HTML to a PDF file

Try It

# Markdown to PDF
curl -X POST https://your-api.onrender.com/md-to-pdf \
  -H "Content-Type: text/plain" \
  -d "# Hello World\n\nThis is a **test**." \
  -o output.pdf

# Markdown to HTML
curl -X POST https://your-api.onrender.com/md-to-html \
  -H "Content-Type: text/plain" \
  -d "# Hello World"

Options

?css=...

Inject custom CSS into the output

?format=A4|Letter|Legal

PDF page format (default: A4)

?margin=20mm

PDF page margins

?standalone=true

Wrap in full HTML document
